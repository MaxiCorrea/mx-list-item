# mx-list-item

## Install

```bash
npm install --save mx-list-item
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'mx-list-item'
import 'mx-list-item/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
